## Privacy

Optional contact information provided by submitters (the free-text "submitted_by" field, which may contain a name, phone number, or other personal contact details) is used internally for contributor recognition (badges, contributor view, duplicate-detection messaging) but is redacted in all public data exports (CSV, JSON, GeoJSON, RAPIDA) to protect submitter privacy. Only whether attribution was provided ("Provided (redacted)" vs. "Anonymous") is included in exported datasets, never the contact details themselves.
